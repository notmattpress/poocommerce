# JavaScript Package Releases

The PooCommerce monorepo includes numerous JavaScript packages to support PooCommerce development, such as `@poocommerce/components`, distributed via NPM. This release process is managed via GitHub Actions.

For the full documentation on how to release a JavaScript package, see the [package release tool documentation](https://github.com/poocommerce/poocommerce/tree/trunk/tools/package-release).
